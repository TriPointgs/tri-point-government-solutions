# Tri-Point Government Solutions LLC - Starter Website

This package includes a clean starter brand and one-page static website.

## Files

- `index.html` — homepage
- `styles.css` — full responsive styling
- `assets/tripoint-logo.svg` — vector logo
- `brand-notes.md` — colors, fonts, and recommended edits

## Before publishing

Replace these placeholders:

- UEI
- CAGE
- Primary NAICS
- Business email
- Phone number
- Contact form backend

## Easy hosting options

### Vercel
1. Upload these files to a GitHub repository.
2. Import the repository into Vercel.
3. Deploy as a static site.

### Netlify
1. Drag the folder into Netlify Drop.
2. Connect a domain later.

### Wix
Use the logo and copy from this package, then rebuild the page visually in Wix.
